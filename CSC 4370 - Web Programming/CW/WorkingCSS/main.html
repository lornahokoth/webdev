Exercise 6

div - div tags should use display: flex. This allows for easily aligning divs next to each other and creating seperate rows. Background color will be rgb(51, 51, 51). The default text color is white.
h1 - we will use the default format for h1 tags.
h2 - h2 tags will use the default format using red text.
table - Tables should be alternating colors and have a width no greater than 70% of the container.
tr - The header background-color will be a dark blue. The rows will alternate between white and a lighter shade of the header color.
th - th should set the font-weight to bold
td - numeric fields should have text-align center. Alpha fields will have text-align set to left.
span - use spans to emphasize important messages. The color of the messages should be red. The font-weight should be bold.
a - a tags should show a description intead of just be the url. https://www.google.com vs Google
p - p tags should be indented using text-indent: 2cm
